# WeChatSwift
iOS WeChat App Written in Swift 5.0 and [Texture](https://github.com/TextureGroup/Texture)


Requirements
==

- [Texture](https://github.com/TextureGroup/Texture)
- Swift 5.0 +
- Xcode 10.2 +


Features
==

- Wechat UI
- Support Text、Image、Voice Message Types
- Moments
